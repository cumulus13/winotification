# icons/

Place your icon files here:

- `icon.ico`  — Used for the Windows system tray (ICO format, 32×32 or 256×256 recommended)
- `icon.png`  — Used for Growl (GNTP) and ntfy icon URL fallback (PNG, 64×64 recommended)

If `icon.ico` is missing, WiNotification falls back to a minimal blank icon so it
still starts cleanly. A 🔔 emoji fallback is shown in the tooltip.

Free tools to create icons:
- https://icofx.ro/
- https://convertico.com/
- https://favicon.io/favicon-generator/
